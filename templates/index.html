deepNets

A framework for deep learning vision tasks , it so easy to use that a simple knowledge of python syntax is enough. Just push the layers in the order of which you want while the rest will be taken care by deepNets.
Simple as that!

Features

Fully connected layers


  • Input layer
  • ReLU layer
  • Batch Normalization layer
  • Layer Normalization layer
  • Loss layer

Convolutional layers


  • Input layer
  • Conv layer
  • Pooling layer
  • Loss layer
  • Trainer

deepNets only uses python ✅ and doesn't depend on other frameworks 😏

Easy installation -- Free to use -- Open Source -- Well takes only 3.9MB 🤯

Installation : pip install deepNets
